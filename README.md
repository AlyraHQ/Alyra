## ALYRA

Alyra is a smart prepaid solar energy platform designed for off-grid rural households. It enables users to purchase electricity units via paycode (usable offline and on feature phones), and the system vends electricity units directly to smart meters.

Alyra predicts energy depletion before outages occur, recommends optimal recharge amounts, supports flexible top-ups, and integrates with Interswitch API for secure transactions.

Alyra makes energy access simple, inclusive, and dependable.

---

## Overview

Access to reliable electricity remains a major challenge in rural and off-grid communities. Alyra bridges this gap by combining prepaid energy vending, predictive intelligence, and secure digital payments into one inclusive platform.

The system is built to:

- Operate in low-connectivity environments
- Support feature phones and offline usage
- Provide intelligent energy consumption insights
- Enable secure and reliable financial transactions

---

## Key Features

### Paycode-Based Electricity Purchase
- Users purchase electricity units using a generated paycode
- Works offline and on feature phones
- No smartphone or constant internet required

### Smart Meter Integration
- Automatically vends purchased units to connected smart meters
- Ensures seamless energy activation

### Predictive Energy Monitoring
- Tracks user consumption patterns
- Predicts energy depletion before outages
- Provides proactive recharge alerts

### Intelligent Recharge Recommendations
- Suggests optimal recharge amounts based on historical usage
- Helps users avoid overpaying or running out unexpectedly

### Flexible Top-Ups
- Supports micro-payments and incremental recharges
- Designed for households with irregular income streams

### Secure Payments
- Integrated with Interswitch API
- Ensures secure, trusted, and reliable transactions

---

## How It Works

1. User purchases electricity units via paycode.
2. Payment is processed securely.
3. Alyra generates and validates the paycode.
4. Units are vended to the user’s smart meter.
5. The system monitors consumption continuously.
6. Users receive recharge recommendations before depletion.
